# Zhihu++

[![GitHub release](https://img.shields.io/github/v/release/zly2006/zhihu-plus-plus)](https://github.com/zly2006/zhihu-plus-plus/releases)

知乎手机客户端，蹲坑神器。去广告，去推广软文，去推销带货，去盐选专栏。

可以设置屏蔽词、AI屏蔽回答、屏蔽用户、屏蔽话题等（以上都是愿景）。
